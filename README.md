# nordicobf
A obfuscator for text.

# prerequisites
Gui.py > GUI for obfuscating text

Deobf.py > GUI for deobfuscating text

Cryptography Package (pip install cryptography)

Python

# info

this obfuscator works by these layers

Text > Base64 encoding > Atbash Cipher > Caeser Cipher > Base85 encoding > Base16 conversion > Icelandic Translation > Swedish Translation > Base12 Conversion > Norwegian Translation

To run:
Open CMD in the directory where you put the GUI.py and Deobf.py

Type: **python gui.py** to open the obfuscator gui

Type: **python deobf.py** to open the deobfuscator gui
